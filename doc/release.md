# 問題点
### リロードが必要なシーンができてしまう
> 回答者のページに移った場合、イラストが揃ってから再レンダリングのためリロードが必要
>> データベース上の更新を検知して再レンダリング

- データが追加されたら、適宜再レンダリングされるように変更
- ロビーページもルームが作成されたら再レンダリングされるように変更

---

### 描いている人は他の描いたものが見られない
> 描画ページから回答ページに移った後だと、ボタンが表示されていないため最初のイラスト以外見られない
>> 他の人も見られるように回答者以外に見えるボタンを表示する

---

### 終了後、リセットして再度ルームに戻る必要がある
> 現状の環境だと、ゲームが終了した後何もできないのでもう一度遊ぼうとした場合、ルームを再生成しないといけない
>> 終了後、お題を変えて再度ルームに再集合できる機能の追加(回答者もリセット)
